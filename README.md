## For anyone downloading the demo

**Mac:** download → unzip → double-click **Oncura Demo**  
**Windows:** download → double-click **Oncura-Demo-Windows.exe**

No Python. No Terminal. No `cd demo`.

If macOS says the app is from an unidentified developer (first time only): **right-click** the app → **Open** → **Open**. That is an Apple security step for free research apps; it is not a bug in the app.
