Oncura Demo — native app install
=================================

MAC
  1. Download Oncura-Demo-mac.zip from GitHub Releases
  2. Unzip
  3. Double-click "Oncura Demo.app"
  4. If blocked: right-click the app -> Open -> Open (once only)

WINDOWS
  1. Download Oncura-Demo-Windows.exe
  2. Double-click to run
  3. If SmartScreen warns: More info -> Run anyway

IN THE APP
  Load sample -> Classify -> read prediction and chart

No Python install required for these downloads.

Research pipeline: https://github.com/rstil2/cancer-alpha
Not for clinical use.
