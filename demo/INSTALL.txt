Oncura Demo — quick install
===========================

WHAT YOU NEED
  Python 3.8 or newer (free): https://www.python.org/downloads/
  Windows: during install, check "Add Python to PATH".

This download is NOT a standalone .exe / .app — Python must be installed once.


WINDOWS — double-click
  1. Unzip Oncura-Demo.zip
  2. Open the Oncura-Demo folder
  3. Double-click:  Start Oncura Demo.bat
  4. Browser opens to http://localhost:8501

  If Windows blocks the file: right-click the .bat → Properties → Unblock → OK,
  then double-click again.

  First run installs Python packages (1–2 minutes).


macOS — double-click
  1. Unzip Oncura-Demo.zip
  2. Open the Oncura-Demo folder
  3. Double-click:  Start Oncura Demo.command
     (Terminal opens and starts the demo)
  4. Browser opens to http://localhost:8501

  First time only: if macOS says the file is from the internet,
  right-click → Open → Open (not double-click the first time).

  Alternative: Terminal → cd to this folder → ./start_demo.sh


LINUX
  Unzip, then in Terminal:
    chmod +x start_demo.sh
    ./start_demo.sh


IN THE APP
  Sidebar → Sample Data → Cancer Sample → Generate Sample Data
