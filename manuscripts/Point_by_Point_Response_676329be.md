## Editorial Comment 1: KEGG Citations

**Comment:** "Please cite the following article(s) when using KEGG:
- Kanehisa, M., Sato, Y., Kawashima, M., Furumichi, M., and Tanabe, M.; KEGG as a reference resource for gene and protein annotation. Nucleic Acids Res. 44, D457-D462 (2016).
- Kanehisa, M. and Goto, S.; KEGG: Kyoto Encyclopedia of Genes and Genomes. Nucleic Acids Res. 28, 27-30 (2000)."

**Response:**

Thank you for this guidance. We have added both requested KEGG citations to our manuscript. These references now appear as citations 77 and 78 in the References section:

> 77. Kanehisa M, Goto S. KEGG: Kyoto Encyclopedia of Genes and Genomes. Nucleic Acids Res. 2000;28(1):27-30.
>
> 78. Kanehisa M, Sato Y, Kawashima M, Furumichi M, Tanabe M. KEGG as a reference resource for gene and protein annotation. Nucleic Acids Res. 2016;44(D1):D457-D462.

The in-text citation has been added in Section 2.4.1 where we describe our use of KEGG cancer pathways (hsa05200 series) for biological pathway integration.

**Comment:** "We noticed that you have included a Data Availability Statement in the manuscript... Kindly follow the below steps to provide same data availability statement in the submission system.
1. Log in to the SNAPP system using your registered credentials.
2. Navigate to the 'Declaration' section.
3. From the list of declaration sections, select the 'Data Availability' section.
4. Click 'Yes' to indicate data availability"

**Response:**

Thank you for this guidance. We have now completed the Data Availability declaration in the SNAPP submission system by:

1. Logging in to the SNAPP system
2. Navigating to the 'Declaration' section
3. Selecting the 'Data Availability' section
4. Clicking 'Yes' to indicate data availability

The Data Availability statement in the submission system now matches the statement in our manuscript.
