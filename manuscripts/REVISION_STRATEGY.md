Multi-cancer genomic classification remains limited by three methodological challenges: effective multi-modal integration, class imbalance without synthetic augmentation, and biologically validated interpretability. We developed a knowledge-guided multi-modal integration framework incorporating biological pathway constraints into feature engineering across six genomic modalities (methylation, mutations, copy number alterations, fragmentomics, clinical features, and ICGC ARGO data). These results demonstrate that knowledge-guided multi-modal integration, combined with balanced real-data design and biologically validated interpretability, provides an effective and transparent approach for multi-cancer genomic classification.

```markdown
We downloaded genomic data from TCGA via the GDC Data Portal (https://portal.gdc.cancer.gov/) using [specify tool: TCGAbiolinks, GDC Data Transfer Tool, etc.]. 

**Data Types Downloaded:**
- DNA methylation: Illumina HumanMethylation450 array (processed beta values)
- Somatic mutations: VCF files from MuTect2 variant calling
- Copy number alterations: GISTIC2 processed seg files  
- Gene expression: HTSeq-Counts normalized to TPM
- Clinical data: Clinical supplement files

**Quality Control:**
- Removed samples with >10% missing feature values
- Excluded samples with conflicting barcodes
- Filtered samples failing TCGA quality metrics
- Verified sample-level concordance across data types

**Preprocessing Steps:**
1. Methylation: Beta values, filtered for CpG sites in gene promoters
2. Mutations: Binary encoding of driver gene mutations, TMB calculation
3. CNA: Segment mean values, focal/arm-level calls
4. Expression: Log2(TPM+1) transformation, batch correction with ComBat-Seq
5. All features: Robust scaling (median, IQR)

**Missing Data Handling:**
- Features with >30% missingness excluded
- Remaining missing values imputed using KNN imputation (k=5)
- Missingness indicator features added for values >5% missing

**Batch Effect Correction:**
- Identified batch effects using PCA analysis and TCGA plate information
- Applied ComBat batch correction to expression data
- Verified correction using PCA plots pre/post correction
```
